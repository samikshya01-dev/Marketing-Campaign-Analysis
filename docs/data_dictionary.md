# Data Dictionary

## Campaign Data

### campaigns Table
| Column | Type | Description |
|--------|------|-------------|
| campaign_id | INT | Primary key, auto-incrementing identifier |
| campaign_name | VARCHAR(100) | Name of the marketing campaign |
| channel | VARCHAR(50) | Marketing channel (Email, Social, etc.) |
| cost | DECIMAL(10,2) | Campaign spend in USD |
| impressions | INT | Number of times the ad was shown |
| clicks | INT | Number of clicks received |
| conversions | INT | Number of successful conversions |
| revenue | DECIMAL(10,2) | Revenue generated in USD |
| date | DATE | Campaign date |

### Derived Metrics
| Metric | Formula | Description |
|--------|---------|-------------|
| CTR | clicks / impressions * 100 | Click-through rate (%) |
| Conversion Rate | conversions / clicks * 100 | Percentage of clicks that convert |
| ROI | (revenue - cost) / cost * 100 | Return on investment (%) |
| ROAS | revenue / cost | Return on ad spend ratio |
| CPC | cost / clicks | Cost per click |
| CPA | cost / conversions | Cost per acquisition |

## Customer Data

### customers Table
| Column | Type | Description |
|--------|------|-------------|
| customer_id | INT | Primary key, auto-incrementing identifier |
| age | INT | Customer age |
| gender | VARCHAR(10) | Customer gender (M/F) |
| country | VARCHAR(50) | Customer country |
| sessions | INT | Number of website sessions |
| avg_session_duration | DECIMAL(10,2) | Average session duration in seconds |
| pages_per_session | DECIMAL(10,2) | Average pages viewed per session |
| transactions | INT | Number of completed purchases |
| revenue | DECIMAL(10,2) | Total revenue generated by customer |

### Customer Segments
| Segment | Description | Criteria |
|---------|-------------|----------|
| High-Value Buyers | Top spending customers | revenue > 66th percentile AND transactions > 3 |
| Deal Seekers | Price-sensitive buyers | transactions > 0 AND revenue <= 66th percentile |
| Casual Browsers | Non-converting visitors | sessions > 0 AND transactions = 0 |

## Channel Categories

### Marketing Channels
| Channel | Description | Example Campaigns |
|---------|-------------|------------------|
| Email | Email marketing campaigns | Newsletter, Promotional emails |
| Paid Search | Search engine advertising | Google Ads, Bing Ads |
| Social Media | Social platform campaigns | Facebook, Instagram ads |
| Display | Display advertising | Banner ads, Remarketing |
| Organic Search | Natural search traffic | SEO efforts |
| Direct | Direct website visits | Type-in traffic |
| Referral | Traffic from other sites | Partner referrals |

### Performance Categories
| Category | Description | Criteria |
|----------|-------------|----------|
| High Performers | Best performing campaigns | ROI >= 200% |
| Good Performers | Above average campaigns | ROI >= 100% |
| Break Even | No profit/loss | ROI >= 0% |
| Loss Making | Unprofitable campaigns | ROI < 0% |

## Data Quality Rules

### Validation Rules
1. Impressions must be >= clicks
2. Clicks must be >= conversions
3. Cost and revenue must be >= 0
4. Sessions must be > 0
5. Date must be within valid range

### Missing Value Handling
| Field | Strategy |
|-------|----------|
| Numeric | Replace with median |
| Categorical | Replace with mode |
| Dates | Remove record |
| Revenue | Remove if critical |
